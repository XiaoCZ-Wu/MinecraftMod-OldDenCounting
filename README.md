# MincraftMod-OldDenCounting

**老登计数法**

**一种新型计数方式**

**本模组仅用作研究、学习使用，不可用于其他用途！**

</div>

# 使用方法

- 将 `odcounting-<Mod版本号>-forge-1.19.2.jar` 放入 `mods` 文件夹中

- 将 `OldDenCounting-<对应Mod版本号>.zip` 数据包放入 `datapacks` 文件夹中
